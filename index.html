Scratch Player 3 | JGames101

INFORMATION

Scratch Player 3 is a Chrome extension that adds a button to Scratch that lets you run a Scratch Project in the new HTML5 player.
It was built using JQuery.

HOW TO

Once you have downloaded the .zip file, extract it, go to "chrome://extensions", Enable developer mode, click load unpacked extension, and select the folder where you extracted it. Confirm install, and you're done! You can now use the extension.
Developer mode must be enabled, or Chrome will disable it after you relaunch your browser.

You can also install it as a userscript to whatever version of greasemonkey/tampermonkey supports your browser.
