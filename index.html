Scratch Player 3 | James M

INFORMATION

Scratch Player 3 is a Chrome extension that adds a button to Scratch that lets you run a Scratch Project in the new HTML5 player.
It was built using JQuery.

HOW TO

Once you have downloaded the .crx file, go to "chrome://extensions", and drag the .crx file from your file browser to that page. Confirm install, and you're done! You can now use the extension.
